Data-Analysis Step-wise approach to data analysis This repo concerns about the stages of various ways for data analysis. It is divided into sub-folders like beginners, intermediate and advanced. The beginner is only being uploaded till now.

BEGINNER: The beginner section is focussed as an initiation to start programming in R and perform basic operations for clustering, classification and other functions.

The contents covered till now:

  • kNN
  • fuzzy c-means
  • normalization

Dataset used: iris preinstalled and readily available in RStudio

or download from: https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data Copy the entire text from this link, paste it in the notepad and save it as iris.csv
